## [1.0.0] - 2025-11-06

### Added
- Initial release of React Modal library
- Web modal component with GSAP animations
- React Native modal component with Reanimated animations
- Support for centered modal and bottom sheet types
- `modal.open()` API for opening modals programmatically
- `useModal()` hook for accessing modal context
- `ModalProvider` component for managing modals
- Customizable styling support (className for web, StyleProp for mobile)
- Click outside to close functionality
- Close button with customizable styling
- TypeScript type definitions
- Full documentation and examples
